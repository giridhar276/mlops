# Dataset Dictionary

`customer_churn_500.csv` has 500 realtime-style customer records.

Columns:
1. customer_id
2. age
3. gender
4. region
5. plan_type
6. tenure_months
7. monthly_charges
8. support_tickets
9. data_usage_gb
10. contract_type
11. churn

Use `churn` as classification target.
Use `monthly_charges` as regression target in regression examples.
